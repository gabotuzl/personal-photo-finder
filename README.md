# personal-photo-finder
This is a project that uses a fine-tuned projection head on top of OpenAI's CLIP ViT-L-14 model to achieve caption-to-image retrieval tasks using user-specific classes.

### Format
This project was developed in a .ipynb notebook, which is seen in the repo. It shows all the steps from data preparation, model selection, projection head fine tuning, testing, evaluation, and drawing of conclusions. This work is part of a group effort.
